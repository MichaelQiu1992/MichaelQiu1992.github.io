Xcode 常用快捷键

发表于 2016-10-22 | 分类于 iOS 开发 | 本文总阅读量 次

工欲善其事必先利其器，想成为一个优秀的 iOS developer 肯定要熟悉 Xcode，熟练使用 Xcode 的快捷键。下面我就以 Xcode8 为例，介绍一下 Xcode 中常用的快捷键。

Mac 键盘图标

Mac 键盘图标及对应的快捷键图标:

  • ⌃ ： control
  • ⌥ ： option (alt)
  • ⌘ ： command
  • ⇧ ： shift
  • ⇪ ： caps lock

Xcode 快捷键

软件快捷键

常用的有：

  • Preferences : ⌘ (command) + ,
  • Quit Xcode : ⌘ (command) + Q

File 类快捷键

常用的有：

  • New -> File : ⌘ (command) + N
  • New -> Project : ⇧ (shift) + ⌘ (command) + N
  • Save : ⌘ (command) + S

Edit 类快捷键

常用的有：

  • Undo : ⌘ (command) + Z
  • Redo : ⇧ (shift) + ⌘ (command) + Z
  • Cut : ⌘ (command) + X
  • Copy : ⌘ (command) + C
  • Paste : ⌘ (command) + V
  • Select All : ⌘ (command) + A

View 类快捷键

Standard Editor 类快捷键

常用的有：

  • Show Document Items : ⌃ (control) + 6

Assistant Editor 类快捷键

常用的有：

  • Show Assistant Editor : ⌥ (option) + ⌘ (command) + ↩︎

Navigator 类快捷键

左侧 Navigator 对应的快捷键为 ⌘ (command) + 1 到 ⌘ (command) + 8，隐藏 Navigator 的快捷键为 ⌘ (command) + 0，其中常用的有：

  • Show Project Navigator : ⌘ (command) + 1
  • Show Find Navigator : ⌘ (command) + 3
  • Show Breakpoint Navigator : ⌘ (command) + 7
  • Hide Navigator : ⌘ (command) + 0

Debug Area 类快捷键

常用的有：

  • Activate Console : ⇧ (shift) + ⌘ (command) + C
  • Hide Debug Area : ⇧ (shift) + ⌘ (command) + Y

Utilities 类快捷键

常用的有：

  • Hide Utilities : ⌃ (control) + ⌘ (command) + 0

Find 类快捷键

常用的有：

  • Find : ⌘ (command) + F
  • Find in Project : ⇧ (shift) + ⌘ (command) + F
  • Find and Replace : ⌥ (option) + ⌘ (command) + F

Navigate 类快捷键

常用的有：

  • Go Forward : ⌃ (control) + ⌘ (command) + →
  • Go Back : ⌃ (control) + ⌘ (command) + ←
  • Jump to Selection : ⇧ (shift) + ⌘ (command) + L
  • Jump to Definition : ⌃ (control) + ⌘ (command) + J
  • Jump to Next Counterpart : ⌃ (control) + ⌘ (command) + ↑
  • Jump to Previous Counterpart : ⌃ (control) + ⌘ (command) + ↓
  • Jump to Line in ... : ⌘ (command) + L

Editor 类快捷键

Structur 类快捷键

常用的有：

  • Shift Right : ⌘ (command) + ]
  • Shift Left : ⌘ (command) + [
  • Move Line Up : ⌥ (option) + ⌘ (command) + [
  • Move Line Down : ⌥ (option) + ⌘ (command) + ]
  • Comment Selection : ⌘ (command) + /
  • Add Documentation : ⌥ (option) + ⌘ (command) + /

Code Folding 类快捷键

常用的有：

  • Fold : ⌥ (option) + ⌘ (command) + ←
  • Unfold : ⌥ (option) + ⌘ (command) + →
  • Fold Methods & Functions : ⌥ (option) + ⇧ (shift) + ⌘ (command) + ←
  • Unfold Methods & Functions : ⌥ (option) + ⇧ (shift) + ⌘ (command) + →

Product 类快捷键

常用的有：

  • Run : ⌘ (command) + R
  • Build : ⌘ (command) + B
  • Clean : ⇧ (shift) + ⌘ (command) + K
  • Stop : ⌘ (command) + .

Debug 类快捷键

常用的有：

  • Breakpoint -> Add Breakpoint at Current Line : ⌘ (command) + \
  • Step Over : Fn + F6
  • Step Into : Fn + F7
  • Step Out : Fn + F8

Source Control 类快捷键

常用的有：

  • Commit : ⌥ (option) + ⌘ (command) + C

Window 快捷键

常用的有：

  • Documentation and API Reference : ⇧ (shift) + ⌘ (command) + 0
