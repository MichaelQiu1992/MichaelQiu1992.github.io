Mac 下使用 Hexo 搭建个人博客

发表于 2016-03-24 | 分类于 Website | | 阅读次数 | 本文总阅读量 次

Hexo 是一个快速、简洁且高效的博客框架， 具有超快速度、一键部署、支持 Markdown、丰富的插件等特点。初见 Hexo 就被它惊艳到，相当简洁，并且有众多持续更新的主题。

自己之前使用 WordPress 完成个人博客的搭建，WordPress 有自己的后台管理系统，可以增加不少小工具和插件。WordPress 是一个强大但相当冗杂的博客管理工具。

考虑得到 Hexo 的惊艳和简洁，决定将个人博客从 WordPress 实现转到 Hexo 实现。

网站搭建步骤

Hexo 安装前准备

Hexo 的官网 已经很详细的介绍了搭建过程。这里，我只是介绍关键步骤以及如何采用 Hexo + GitHub Page 和 Hexo + Coding 实现个人博客的创建。

需要在电脑安装下面两个软件。

  • Node.js
    Node.js 用来生成静态页面,到 Node.js 官网 下载最新版本，一直点继续即可。

  • Git
    Git 用来将本地 Hexo 内容提交到 Github 上，到 Git 官网 下载最新版本，一直点继续即可。

绑定个人域名

购买域名

购买域名，可以到 GoDaddy 官网，也可以到阿里万网购买。

添加 CNAME 文件

在 /blog/source 目录下新建文件名为： CNAME 文件，注意没有后缀名！直接将自己的域名如：swifter.wang 写入。

域名解析

登录你注册域名的网站，比如万网，在你购买的域名后边点击：解析 –> 添加解析
记录类型：CNAME
主机记录：填写@
记录值：michaelqiu1992.github.io. (不要忘记最后的.，michaelqiu1992改为你自己的用户名)，点击保存即可。

此时，大功告成！

注意点

  • 在配置所有的 _config.yml 文件时（包括 theme 中的），在所有的冒号:后边都要加一个空格，否则执行 hexo 命令会报错
